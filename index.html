Thumper Game and Tooling Documentation

This is a central place to host all up to date documentation related to Thumper. This includes all the tooling around Thumper and includes a lot of low level information about the file specifications of Thumper.

Motivation

Currently all the documentation surrounding Thumper and it's tools are scattered all over the place. A lot of which is outdated and incorrect information. This wiki will serve as a central location for all documentation surrounding Thumper. We are activly gathering all existing information and migrating to this wiki.

Open Source and Open Contribution

This document will happily accept contributions and updates as new information is discovered or documentation generally needs changes.
